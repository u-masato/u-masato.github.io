Chrome Extension

site-dock

サイト（eTLD+1）ごとに「よく使うリンク」を保存し、拡張アイコンを押すだけで一覧からすぐ移動できる Chrome 拡張機能です。

主な機能

  • サイト（eTLD+1）ごとにリンクを保存
  • 拡張アイコン（ワンボタン）で、そのサイトのリンク一覧を表示
  • リンクはボタンで選択して同一タブ遷移
  • リンク名の編集、不要なリンクの削除

想定ユースケース

  • 管理画面、ログアウト、ユーザー一覧などへの素早い移動
  • 同じサイト内の特定ページを行き来する手間の削減

サポート

連絡先: GitHub Issues

Privacy / プライバシー

プライバシーポリシー

Limited Use

site-dock は Chrome Web Store User Data Policy の Limited Use 要件に準拠し、ユーザーデータを中核機能の提供に必要な範囲でのみ使用します。
