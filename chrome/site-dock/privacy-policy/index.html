Privacy Policy / プライバシーポリシー（site-dock）

発効日: 2026-02-07

提供者: u-masato

連絡先: GitHub Issues

1. 本ポリシーの対象

本ポリシーは、Chrome拡張機能「site-dock」（以下「本製品」）における、ユーザーデータの取り扱いを定めるものです。

2. 収集・保存するデータ

本製品は、ユーザーが保存した以下の情報を保持します。

  • リンク名（ユーザーが付ける表示名）
  • リンクURL

本製品は、上記以外のデータ（閲覧履歴、認証情報、位置情報、個人の連絡先情報など）を収集しません。

3. 利用目的

  • サイト（origin）ごとにリンク一覧を表示し、ユーザーがワンクリックで遷移できるようにするため
  • リンク名の編集、リンクの削除など、ユーザー操作を可能にするため

4. データの共有・第三者提供

本製品は、保存データを外部サーバーへ送信しません。保存データを第三者へ販売・貸与・共有しません。

ただし、ユーザーがChrome同期を有効にしている場合、保存データはChromeの同期機能により、Googleアカウントに紐づく同期範囲で複数端末に同期されることがあります。

5. 保存場所と保持期間

  • 保存場所: Chrome拡張機能の保存領域（chrome.storage.sync）
  • 保持期間: ユーザーが削除するまで保持されます

6. ユーザーによる削除方法

ユーザーは本製品のポップアップ画面からリンクを削除できます。また、Chromeの拡張機能管理画面から本製品を削除することで、保存データも削除されます（Chrome同期の挙動はユーザー設定に依存します）。

7. セキュリティ

本製品は、保存データを本製品の機能提供に必要な範囲でのみ取り扱います。外部サーバー送信を行わない設計により、外部への漏えいリスクを最小化します。

8. Limited Use（Chrome Web Store User Data Policy への準拠）

本製品は Chrome Web Store User Data Policy の Limited Use 要件に準拠し、ユーザーデータを本製品の中核機能（サイトごとのリンク保存・表示・遷移）の提供に必要な範囲でのみ使用します。

9. 改定

本ポリシーの内容は、法令・ポリシー変更や機能追加に応じて改定される場合があります。改定後の内容は、以下の公開ページにて公表します。
